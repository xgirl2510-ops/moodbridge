# MoodBridge - ClaudeKit Project Instructions

## Project Overview

MoodBridge is a Flutter/Firebase mobile app that connects happy users with sad users through anonymous encouragement messages.

## Tech Stack

- **Frontend:** Flutter + Dart + Riverpod (state management)
- **Backend:** Firebase (Auth, Firestore, Storage, Cloud Functions, FCM)
- **Analytics:** Firebase Analytics + Crashlytics

## Development Guidelines

- Follow workflows defined in `.claude/rules/`
- Always read `./README.md` before planning implementations
- Activate relevant skills from the skills catalog during tasks
- Strictly follow `.claude/rules/development-rules.md`
- Use `.claude/skills/.venv/bin/python3` when running Python skill scripts

## Key Directories

- `flutter/` - Flutter app source code
- `firebase/` - Firebase configuration and Cloud Functions
- `plans/` - Implementation plans
- `docs/` - Project documentation
- `.claude/rules/` - Development workflows and rules
- `.claude/skills/` - ClaudeKit skills and scripts
- `.claude/agents/` - ClaudeKit agent definitions
